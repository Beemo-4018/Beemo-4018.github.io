🧠 DICOM & MRI 복원 AI 파이프라인

2026.03 · PyTorch · pydicom · h5py · UNet
Overview

실제 CT DICOM 데이터와 fastMRI 무릎 MRI 데이터로 SwiftMR과 동일한 구조의 k-space 언더샘플링 → UNet 복원 파이프라인을 end-to-end로 구현했습니다. MSE Loss의 엣지 뭉개짐 문제를 실험으로 직접 확인하고 SSIM Loss를 결합해 개선했습니다.

성과
+2.04 dBPSNR 향상
+0.119SSIM 향상
↓ 79%CT Loss 감소
주요 역할
  • DICOM 표준(Transfer Syntax, UID 계층, HU 변환, Windowing) 학습 및 pydicom 기반 CT 전처리 파이프라인 설계
  • 189개 DICOM 슬라이스 → 3D 볼륨(189×512×512) 구성 및 PyTorch DataLoader 연결
  • fastMRI 무릎 MRI 데이터셋으로 k-space 언더샘플링(가속도 4배) → UNet 복원 구현
  • MSE Loss 단독 사용 시 엣지 뭉개짐 문제 실험으로 확인 → SSIM Loss 결합 Combined Loss로 개선
  • PSNR / SSIM 기반 정량 평가 체계 수립
인사이트
  • 물리치료사로 MRI 촬영 과정을 직접 경험한 배경으로 언더샘플링 복원의 임상적 의미(환자 불편 감소, 촬영 시간 단축)를 명확히 인지하며 개발
  • SwiftMR이 푸는 문제와 완전히 동일한 태스크 구조 — k-space 25% 샘플링으로 촬영 4배 빠르게

🚭 흡연 데이터 분석 건강 인사이트 도출

2025.12 · 팀 코딩맨 (5인, 팀장) · AI 헬스케어 프로젝트
Overview

흡연 공공데이터 기반으로 심혈관 질환 위험 요인을 분석하고, 데이터 기반 공중보건 정책을 제안했습니다. 5인 팀의 팀장으로 프로젝트를 주도하고 발표까지 담당했습니다.

성과
×16복합 위험도 증가
ROI 1.8배예방 정책 효과
🏆최우수상 수상
주요 역할
  • 팀장으로서 가설 3개 설계 및 프로젝트 방향 설정, 팀원 역할 분배, 발표 담당
  • 가설 1 — 흡연 여부에 따른 건강 지표 차이 검증
  • 가설 2 — 나이대 × 체중상태 16개 그룹 교차분석 (30대 저체중 흡연율 54.5% vs 70대 비만 3.1%)
  • 가설 3 — 4가지 복합 위험 요인 분석 (보유 수마다 위험도 2배씩 증가 → 4개 보유 시 16배)
  • 팀원 비협조 상황에서 4인 역할 재조율하여 일정 내 완수
인사이트
  • 개인 작업보다 팀 작업이 더 높은 완성도의 결과물을 낼 수 있다는 것을 체득
  • 팀원 의견 수렴·조율 과정에서 리더십과 커뮤니케이션 역량 발휘
